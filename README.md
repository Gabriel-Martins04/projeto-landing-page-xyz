# Landing Page Agência XWY

Este projeto é um modelo de **Landing Page** desenvolvido utilizando apenas **HTML e CSS**.

O objetivo foi praticar a criação de layouts modernos, organização de seções e estilização de páginas voltadas para apresentação de produtos ou serviços.

## 📌 Sobre o projeto

A página apresenta uma estrutura comum de **landing pages**, com seções organizadas para destacar informações importantes e conteúdo visual.

Este projeto foi desenvolvido como parte do curso DevQuest, módulo de **desenvolvimento front-end**.

## 🚀 Tecnologias utilizadas

- HTML
- CSS

## 🎯 Objetivos do projeto

- Praticar a criação de layouts de landing page
- Melhorar habilidades de estilização com CSS
- Trabalhar organização e estrutura de páginas
